Privacy Policy – maelShine Bot

Last Updated: February 26, 2026

1. Introduction

This Privacy Policy explains how maelShine ("the Bot") collects, uses, stores, and protects information when used within the Discord platform. By inviting or using the Bot, you agree to the practices described in this Policy.

2. Service Description

maelShine Bot is an independent interactive Discord application developed and operated by the maelShine Development Team. It simulates a lightweight, turn-based RPG experience within Discord servers.

The Bot is not a standalone commercial video game. It functions as a Discord application that provides simple gameplay mechanics through button interactions, message components, and automated responses.

3. Independent Service

maelShine Bot is not affiliated with, endorsed by, or sponsored by Discord Inc. All trademarks belong to their respective owners.

4. Information Collected

The Bot may collect and process the following minimal data:

  • Discord User ID
  • Discord Username and Display Name
  • Server (Guild) ID
  • Gameplay-related data (rank, inventory, progression, statistics)
  • Messages sent directly to the Bot for command processing

The data collected is strictly limited to what is necessary for operating the RPG mechanics. The Bot does not intentionally collect sensitive personal data such as legal names, physical addresses, financial information, passwords, or government identification.

5. Purpose of Data Processing

Data is processed exclusively for:

  • Operating turn-based RPG mechanics
  • Maintaining player progression and inventory systems
  • Operating ranking systems as a gameplay feature
  • Preventing abuse or exploitation
  • Improving service stability and functionality

Data is stored in a locally hosted database environment maintained by the maelShine Development Team. Reasonable administrative and technical safeguards are implemented to reduce risks of unauthorized access, misuse, or data exposure.

While good-faith security practices are applied, no storage system can be guaranteed to be completely secure. By using the Bot, users acknowledge and accept this inherent limitation.

7. Data Sharing

8. Public Gameplay Information

As part of the gameplay mechanics, non-sensitive information such as usernames and ranking positions may be displayed publicly within Discord servers. The ranking system is a gameplay feature designed solely to reflect player progression and performance.
